Objetivo General del OVI

El presente curso se propone instruir y demostrar el proceso a seguir en la vinculación de una base de datos (SQL Server o MySql) con un sitio web diseñado sobre lenguaje PHP. También se propone demostrar de manera práctica y acertada el resultado de la vinculación mencionada. Lo anterior mediante la creación de una interfaz web que permita la consulta, la edición, modificación y eliminación de información obtenida de las consultas que la aplicación realice a la base de datos correspondiente.

Objetivos Especificos

Objetivo específico 1: Diseñar el código en PHP que demuestre la vinculación con un motor de Base de Datos en Mysql o SQL Server, realizando las validaciones referentes a una correcta asociación con el Motor. Este paso contemplará también la creación de la base de datos, sus tablas y campos. Objetivo específico 2: Diseñar gráficamente la interfaz que permitirá realizar operaciones de consulta, edición, modificación y eliminación de la información obtenida de la Base de Datos. Objetivo específico 3: Desarrollar el correspondiente código en PHP que dará uso práctico y vinculará a la interfaz creada con la información contenida en la Base de Datos.
